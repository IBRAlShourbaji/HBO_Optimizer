# Heavy Ball Optimizer (HBO)

Source code for the **Heavy Ball Optimizer** algorithm.

### Paper
This code implements the metaheuristic algorithm presented in the paper:
*"Heavy Ball Optimizer: A Momentum-Driven Metaheuristic with Superior Scalability and Real-World Applications"*

### How to run
1. Ensure you have MATLAB installed.
2. Download all files in this repository.
3. Open `Main.m` in MATLAB.
4. Run the script.
